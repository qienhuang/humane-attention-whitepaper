# Humane Attention Whitepaper

This repository contains a practical whitepaper on the governance of human
attention in modern smartphone-centered systems.

It focuses on design principles, system-level interventions, and individual
practices that aim to restore human agency over attention.

This work **uses** the FIT (Force–Information–Time) framework as one analytical
lens, but does **not** represent, extend, or redefine the FIT framework itself.

---

## Status

- Document type: Applied whitepaper
- Scope: Smartphone and attention-driven digital systems
- Nature: Normative and design-oriented
- Relationship to FIT: External application
